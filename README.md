# pftracker
Personal Finance Tracker built using React and TS

## Features
- Dashboard summary with income, expense, and balance
- Category-wise expense distribution
- Monthly income vs expense charts
- Budget tracking visualization

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- 
## Learning Outcome
- Understanding real-world React project structure
- Working with reusable components
- Handling charts and UI state effectively
